👨‍🍳 Mise en Place

App collaborative cuisine — configuration Supabase

1. Créer la table Supabase 
create table if not exists mise_en_place (
  id uuid default gen_random_uuid() primary key,
  section text not null,
  task_name text not null,
  status text not null default 'todo',
  target_qty int not null default 0,
  done_qty int not null default 0,
  created_at timestamptz default now()
);
alter table mise_en_place enable row level security;
create policy "public_all" on mise_en_place
  for all to anon using (true) with check (true);

Supabase → SQL Editor → coller et exécuter

2. Connecter le projet

Supabase → Settings → API → Project URL + anon key
